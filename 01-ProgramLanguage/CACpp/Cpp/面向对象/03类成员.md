内联成员函数
内联函数可以提高函数运行效率。这是因为内联函数在编译的时将代码直接嵌入到调用的地方，从而减少了函数调用的开销。
缺点：内联函数每调用一次，就会将代码嵌入到对应的地方，导致程序体积增大(典型的以空间换时间)
注意，内联函数仅仅是给编译器一个提示。因为编译器不会将带有 switch、for 等语句或者大段的代码视为内联函数。
内联函数有两种定义方式： 1. 使用inline关键字  2. 在类中写实现代码
```c++
class cName {
private:
    inline inlFunc1();
    void inlFunc2(){
        statements;
    }
};
```
