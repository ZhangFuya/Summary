# Qt中的QLabel
## Inheritance  Relationship
```c++
Inherits: QFrame
```

## Introduce
QLabel(标签)，用于显示文本或图片。
QLabel中可以显示的类型有：纯文本(QString)、富文本(QString)、图像(QPixmap)、动画(QMovie)、数字 以及 空类型。
